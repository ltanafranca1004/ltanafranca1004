Computing Science student at SFU building things that help people.

Currently building the web platform for [Unify Social](https://unifysocial.ca) — a settlement app for newcomers to Canada.

### What I'm building

- 🧡 **[Unify Social](https://unifysocial.ca)** — web platform for newcomers settling in Canada. Built with Next.js, TypeScript, and Supabase
- 🌐 **[Unify Landing Page](https://unifysocial.ca)** — marketing site built with Astro on Cloudflare

### Other projects

- 🐢 **Turtle Trips AI** — AI travel itinerary planner built at Stormhacks 2025
- 🥗 **NutriFit** — full-stack nutrition planner with Java, PostgreSQL, and Docker
- 🗺️ **Vancouver Amenity Recommender** — ML-based recommendation system using embeddings and OpenStreetMap
